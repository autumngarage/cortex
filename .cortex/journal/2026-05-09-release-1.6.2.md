# Cortex v1.6.2 released — context integrity roadmap + token budget guardrails

**Date:** 2026-05-09
**Type:** release

> Cortex v1.6.2 ships the first context-integrity roadmap artifacts and the

> agent-facing memory protocol.

- **Kind:** GitHub Release + Homebrew tap formula bump

## What shipped

- `docs: define cortex context integrity roadmap` (#255) records the product
  direction for context packs, importers, drift checks, and production-ready
  context assembly.
- `feat(manifest): expose token budget guardrails` (#257) adds manifest metrics
  for token budgets, per-entry token estimates, omitted hot files, stale items,
  next optional files, and first CLI wiring through `cortex status`.
- `docs(journal): auto-draft pr-merged entry for #255` (#256) and
  `docs(journal): auto-draft pr-merged entry for #257` (#258) keep Cortex's
  journal aligned with the shipped roadmap and instrumentation work.
