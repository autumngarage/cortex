## What shipped

- **#196** — `cortex journal draft release` auto-substitutes the T1.10 template from git tag + `gh release view` metadata + PR subjects since the previous tag. Closes #192.
- **#197** — `cortex sync` umbrella command (refresh-state + refresh-index + doctor) plus auto-sync on minor version-bump via `.cortex/.last-cli-version` marker. Closes #190.
- **#199** — `cortex check-triggers --since <ref> | --staged` evaluates Tier 1 triggers (T1.1, T1.4, T1.5, T1.8) against a diff and emits NDJSON describing every trigger that fired. Foundational primitive for the rest of the Tier 1 automation roadmap. Closes #195.
- **#200** — Post-merge `cortex-pr-merged-hook` no longer commits directly to `main`; ships auto-drafts via a `docs/journal-pr-<N>` feature branch and `gh pr merge --auto`. Recursion guard via commit-subject pattern. Closes #193 and #194.

Plus the homebrew-cortex tap fix from earlier in the day (PR #2 in that repo, closes #134): caveats no longer pin specific SPEC/Protocol versions; future releases don't need a tap edit to keep the install message truthful.
