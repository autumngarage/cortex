## What shipped

- `cortex update` is now the canonical umbrella command for bringing a repo's `.cortex/` up to date.
- `cortex update --check` verifies generated layers without writing, including stale source hashes and `state.md` Generator-version drift.
- `cortex sync` still works as a deprecated alias and prints a one-line warning pointing users to `cortex update`.
- `refresh-state`, `refresh-index`, and `doctor` remain independently documented primitives.
- README command docs now list `update` as primary and `sync` as deprecated.
- The manifest Journal omission regression from cortex#239 was fixed while verifying the release branch.
- The README now carries the Autumn Garage attribution line from PR #236.
