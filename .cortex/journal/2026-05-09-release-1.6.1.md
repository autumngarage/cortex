## What shipped

- `fix(hook): isolate pr-merged journal drafts` (#250) keeps the post-merge
  hook from leaving draft Journal state on `main`, skips recursive auto-sync for
  `check-triggers`, and records recovery-branch guidance for interrupted drafts.
- `docs(protocol): define lookup-first context reads` (#251) makes hot/cold
  context lookup order explicit: manifest/hot files first, `cortex grep`/`rg`
  next, `cortex retrieve` only for targeted expansion.
- `feat(retrieve): add agent compact output` (#252) adds
  `cortex retrieve --for-agent`, a citation-first JSON shape with bounded
  excerpts and next-step guidance for agents.
- `feat(retrieve): make semantic backfill opt-in` (#253) keeps BM25 as the
  low-cost default, requires explicit semantic embedding builds, and detects
  missing, partial, or orphaned vector rows before semantic/hybrid retrieval.
